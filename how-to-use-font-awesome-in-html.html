How To Use Font Awesome In HTML

Font Awesome is a font and icon based on CSS. To use the Font Awesome icons, you need to add a cdn URL in the <head> section of your HTML page. You don't need to download and install only need to add the CDN URL inside the <head> section.

You can go the Font Awesome site fontawesome.com to find out all the free icons.

  1. Font Awesome is a font and icon based on CSS.
  2. Need to add a cdn URL in the <head> section of your HTML page.
  3. You don't need to download and install only need to add the CDN URL inside the <head> section.
  4. Go the Font Awesome site fontawesome.com to find out all the free icons.

CDN URL:

<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/4.7.0/css/font-awesome.min.css">

Example Of Using Font Awesome in HTML:

<!DOCTYPE html>
<html>
    <head>
    <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/4.7.0/css/font-awesome.min.css">
    </head>
    <body>

        <i class="fa-brands fa-youtube"></i>
        <i class="fa-brands fa-youtube" style="color:red;"></i>
        <i class="fa-brands fa-youtube" style="color:red;font-size:20px;"></i>
        <i class="fa-brands fa-youtube" style="color:red;font-size:30px;"></i>

        <i class="fa fa-regular fa-angles-right"></i>
        <i class="fa fa-duotone fa-angles-right" style="color:red;"></i>
        <i class="fa fa-angle-double-right" style="color:red;font-size:20px;"></i>
        <i class="fa fa-angle-double-right" style="color:red;font-size:30px;"></i>

        <i class="fa-solid fa-house"></i>
        <i class="fa-solid fa-house" style="color:red;"></i>
        <i class="fa-solid fa-house" style="color:red;font-size:20px;"></i>
        <i class="fa-solid fa-house" style="color:red;font-size:30px;"></i>
    
        <i class="fa-solid fa-cart-shopping"></i>
        <i class="fa-solid fa-cart-shopping" style="color:red;"></i>
        <i class="fa-solid fa-cart-shopping" style="color:red;font-size:20px;"></i>
        <i class="fa-solid fa-cart-shopping" style="color:red;font-size:30px;"></i>
        

</body>
