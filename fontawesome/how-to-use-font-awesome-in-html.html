- How To Use Font Awesome In HTML

Font Awesome is a font and icon based on CSS. To use the Font Awesome icons, you need to add a cdn URL in the <head> section of your HTML page. You don't need to download and install only need to add the CDN URL inside the <head> section.

You can go the Font Awesome site fontawesome.com to find out all the free icons.
